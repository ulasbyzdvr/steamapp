# Steam Free Game Auto Claimer - ELECTRON APP

## Proje Tanımı
Steam üzerindeki ücretsiz oyunları kütüphaneye otomatik olarak ekleyen Electron masaüstü uygulaması.

## Özellikler
- ✅ Steam'e giriş yapabilme
- ✅ Ücretsiz oyunları tespit etme
- ✅ Tespit edilen oyunları hesaba ekleme
- ✅ Modern React UI
- ✅ Sistem tray desteği
- ✅ Arka planda çalışabilme
- ✅ Windows/Mac/Linux destek

## Electron Özellikleri
- Main process: Backend servisi entegre
- Renderer process: React frontend
- Sistem tray integration
- Native menü ve kısayollar
- Minimize to tray
- Auto-start seçeneği (gelecekte)
